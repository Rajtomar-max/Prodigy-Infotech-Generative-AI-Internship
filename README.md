# Generative AI Internship Projects – Prodigy InfoTech

##  About

This repository contains all the projects completed during my **Generative AI Internship** at **Prodigy InfoTech**.

Throughout this internship, I worked on different Generative AI concepts including text generation, image generation, image-to-image translation, neural style transfer, and probabilistic language models. These projects helped me understand both the theoretical concepts and practical implementation of modern AI models.

---

#  Projects

## Task 1 – Text Generation with GPT-2

### 📖 Overview
Built a text generation model using **GPT-2** to generate coherent and contextually relevant text from a given prompt. This project demonstrates how transformer-based language models can generate human-like text after fine-tuning on custom datasets.

### Key Concepts
- GPT-2
- Transformer Architecture
- Fine-tuning
- Natural Language Processing (NLP)
- Text Generation

---

## Task 2 – Image Generation with Pre-trained Models

### 📖 Overview
Implemented an image generation system using pre-trained generative models such as **Stable Diffusion** and **DALL·E Mini** to create images from text prompts.

### Key Concepts
- Stable Diffusion
- DALL·E
- Text-to-Image Generation
- Diffusion Models
- Prompt Engineering

---

##  Task 3 – Text Generation with Markov Chains

### 📖 Overview
Developed a statistical text generation model using **Markov Chains** that predicts the next word or character based on previously observed sequences.

### Key Concepts
- Markov Chains
- Probability Models
- Statistical Language Models
- Text Prediction

---

##  Task 4 – Image-to-Image Translation with cGAN (Pix2Pix)

### 📖 Overview
Implemented an image-to-image translation model using **Conditional Generative Adversarial Networks (cGANs)** and the **Pix2Pix** architecture.

### Key Concepts
- Pix2Pix
- Conditional GAN (cGAN)
- Image Translation
- Computer Vision
- Deep Learning

---

## Task 5 – Neural Style Transfer

###  Overview
Applied Neural Style Transfer** to combine the artistic style of one image with the content of another image using deep neural networks.

### Key Concepts
- Neural Style Transfer
- Convolutional Neural Networks (CNN)
- Feature Extraction
- Image Processing
